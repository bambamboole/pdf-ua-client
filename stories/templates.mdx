# Templates & Blocks

A template is a versioned document made of **rows**, and each row holds one or
more **blocks**. Blocks carry `config` (layout, styling, and block-specific options such as a heading's level) and bind to runtime
`data` by `id`.

```json
{
  "version": 1,
  "config": {},
  "rows": [{ "blocks": [{ "type": "heading", "id": "title", "config": { "level": 1 } }] }]
}
```

## Rows and columns

A row with multiple blocks lays them out as columns; `columnWidths` controls the
relative widths. Data is bound per block via its `id`.
